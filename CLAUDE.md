# Push Box (Sokoban) Game

A terminal-based puzzle game built with Python.

## Project Overview
- Simple Sokoban clone using ASCII graphics
- Runs in terminal with keyboard controls (WASD or arrow keys)
- Single level puzzle

## How to Run
```
python3 pushbox.py
```

## Game Controls
- WASD or Arrow Keys: Move player
- R: Restart level
- Q: Quit

## Symbols
- `#` Wall
- `@` Player
- `$` Box
- `.` Target
- `*` Box on target
- `+` Player on target
